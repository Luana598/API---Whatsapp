 # Whatsapp API

 ### Sobre o projeto
 Desenvolvida em Node.js, a API utiliza dados armazenados em um arquivo local e oferece rotas para buscar conversas e filtrar mensagens por palavra-chave.

 ### tecnologias utilizadas
- Node.js
- Express.js
